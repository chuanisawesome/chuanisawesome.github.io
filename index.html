We'll be back LED SIGN

Being a student associate at CoMotion Startup Hall we'd always have to put up a sign everytime we leave the table; that's when Cody and I decided to make a LED sign. We use a arduino and a LED matrix to make the sign, and hooked up a on and off switch that makes it easier rather than to unplug usb.

JavaFX Snake Game

My last project for my CSC143 class was to create a Snake Game using JavaFX. This game was build from scratch it took me about 20hrs to build. During my time building this game, I ran into troubles with my code, and was not able to fix it. I went to my Professor's office hours everyday to get help. When I was able to to fix my bugs and make the snake move, I was able to help other students in the class who were struggling. Building the game was tough, but I am glad that I was able to push through, to make my game working as well as helping my fellow classmates.
